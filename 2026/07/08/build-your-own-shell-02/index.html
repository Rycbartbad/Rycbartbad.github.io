我们平时在 Shell 中使用的 ls、cat 等命令都是外部命令，我们在控制台输入 “which xxx”，可以定位 xxx 命令的具体路径。以 Windows 为例：

我这里因为安装了 w64devkit，所以能够正常使用 ls、cat 等命令，所以以上输出可能在你的电脑上不会出现。

那么 Shell 是怎么找到这些命令的路径并调用的呢？

环境变量

对于外部命令，Shell 会借助 PATH 环境变量进行定位。PATH 变量储存了一系列目录路径，各路径之间通过 ; 进行分隔（Linux 下通过 : 进行分隔）。Shell 会按照 PATH 中目录的顺序进行查找，一旦在某一目录下找到了对应的可执行文件，就直接调用该路径下的命令。使用 “echo $env:PATH” 可以打印环境变量。

在 Linux 环境下或者使用 Bash 时，可以使用 “echo $PATH” 打印环境变量
